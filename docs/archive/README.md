# docs/archive/ — 歴史資料の扱い

このディレクトリは、移行作業や過去設計の記録を保管する場所です。現行仕様の正本ではありません。

## 現行仕様を確認する場所

現行仕様や AI の作業手順は `docs/context/` 配下の正本を参照してください。

- `docs/context/OVERVIEW.md`: 現行の全体像と導線。
- `docs/context/SPEC_NEXTJS.md`: Next.js 画面・状態の挙動。
- `docs/context/AI_WORKFLOW.md`: AI の作業手順、読む順、止まる条件。
- `docs/context/TESTING.md`: 品質ゲートと回帰確認。
- `docs/context/REPO_STRUCTURE.md`: リポジトリ構成と整理ルール。

## 読み方

- ここにある文書は、移行判断や過去の設計意図を追うための参考資料です。
- 現行の画面挙動、API、DB、テスト方針と食い違う場合は、`docs/context/` の正本を優先してください。
- ここから現行仕様を復元する場合は、必ず正本とコードの両方で確認してください。
